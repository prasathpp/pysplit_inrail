━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PyInRail is a sophisticated Python application that helps users find train 
tickets on Indian Railways when direct booking is unavailable.

THE MAIN FEATURE: SPLIT JOURNEY SEARCH 🎫

When a train shows "No Availability" for your direct route (e.g., Delhi → 
Bangalore), the split journey feature automatically:

1. Breaks the journey into smaller segments
2. Checks availability on each segment separately
3. Finds combinations where seats are available
4. Books multiple segment tickets on the SAME train

Result: You get your ticket by splitting a long journey into 2-4 segments!

SCENARIO: Want to travel Chennai → Mysore on July 31, 2026

TRADITIONAL APPROACH:
  ❌ Search Chennai → Mysore
  ❌ Status: "NO AVAILABILITY"
  ❌ Keep refreshing all day...
  ❌ Finally give up or overpay for alternate transport

PYINRAIL SPLIT JOURNEY APPROACH:
  ✅ python rail_scrapper.py split --train 16021 --from MAS --to MYS --date 31-07-2026
  
  Algorithm finds:
    OPTION 1 (2 Segments):
      ✅ MAS → KPD: Available (Class SL)
      ✅ KPD → MYS: Available (Class SL)
      → You're booked! Same train, same class, no transfers needed!
    
    OPTION 2 (3 Segments):
      ✅ MAS → BWT: Available
      ✅ BWT → KPD: Available
      ✅ KPD → MYS: Available
      → Alternative if first option fails

Result: In seconds, you have multiple viable booking options!
